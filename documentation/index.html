Documentation

This file contains documentation about how to configure and use this engine for games. It doesn't include every variable or method for each class listed, as some are currently only used by the engine itself. Feel free to make any changes to the engine as required.


Config Files

globals.json

Includes all game config

keybinds.json

Config for all keybinds your game will use, the globals checkKeyBinds method will use these for example to check for action keypresses (like quit / debug button/s).

events.json

You can use this to configure game event triggers (such as clicking a node) to certain actions (like updating a variable).

nodes.json

Game nodes to make available for the engine to use, nodes can be loaded into the game using engine:loadNode(handle). Nodes not added to this list can still be added in manually by creating a node.

For example;

engine:addNode("handle", {x = 0, y = 0, w = 100, h = 100})

engine:loadNode("handle")
